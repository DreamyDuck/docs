# Kmbox B+ Configuration

## Product Overview

<Note>
The Kmbox B+ is our standard keyboard and mouse adapter solution, offering precise control and customization options.
</Note>

## Features

<CardGroup>
  <Card title="Connectivity" icon="plug">

    - COM port support
    - Customizable bitrate
  </Card>
  
  <Card title="Compatibility" icon="check">

    - Adjustable response curves

  </Card>

## Setup Instructions

<Steps>
  1. Physical Connection
     Connect your keyboard and mouse to the labeled USB ports
     
  2. Driver Installation
     Install the required Kmbox drivers
     
  3. COM Port Configuration
     Set the correct COM port and bitrate
     
  4. Profile Setup
     Configure your input mapping profile
</Steps>

## COM Port Settings

<Accordion title="Changing COM Port & Bitrate">
### Configuration Steps
1. Open Device Manager
2. Locate Kmbox B+ device
3. Adjust COM port settings
4. Set appropriate bitrate

<Warning>
Incorrect COM port settings may cause connection issues
</Warning>
</Accordion>

## Firmware Management

<Steps>
  1. Download Firmware
     Get the latest firmware from our website
     
  2. Enter Update Mode
     Press and hold the reset button while connecting
     
  3. Flash Firmware
     Use the Kmbox Updater tool
     
  4. Verify Update
     Check firmware version after reboot
</Steps>
