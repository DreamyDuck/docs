# Install Network Adapter Driver
1. After plugging the cable to the Net port, a disk will appear as shown in the picture below.
2. Double-click the disk and run the exe file inside.
3. If no disk to be found, download the driver mannually.

# Check if ping is successful
1. Run `cmd.exe`
<Frame>
      <img src="/en/images/net6.png" />
</Frame>
2. Check if the IP on the kmbox could be successfully pigged.  
```ping 192.168.x.x(IP of your kmbox)```
<Frame>
      <img src="/en/images/net7.png" />
</Frame>

3. If the above is shown, the box is ready. Now you can refer to your dma software for further instructions.
