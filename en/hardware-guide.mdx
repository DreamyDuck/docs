# Hardware Setup Guide

## DMA Board Setup

<Tabs>
  <Tab title="Basic Setup">
    ### Initial Setup
    1. Install the DMA board in an available PCIe slot
    2. Connect the data cable between primary and secondary computers
    3. Verify power connections if using external PSU
    4. Install required drivers and software
    
    <Tip>
    For detailed setup instructions, see our [Setup Guide](/hardware-guide/setup-instruction)
    </Tip>
  </Tab>
  
  <Tab title="Speed Testing">
    ### Performance Testing
    - Run the DMA speed test utility
    - Verify connection stability
    - Check for optimal PCIe lane configuration
    
    <Note>
    Recommended minimum speed: 100MB/s for optimal performance
    </Note>
  </Tab>
</Tabs>
