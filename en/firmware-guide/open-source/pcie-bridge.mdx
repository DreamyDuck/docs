# Firmware PCIe Bridge / CPU Bridge

Firmware PCIe Bridge / CPU Bridge emulates genuine PCIe and CPU bridge controllers. This firmware solution provides comprehensive emulation of system interconnect hardware, ensuring that system and anti-cheat solutions recognize it as legitimate bridge equipment while maintaining a low detection profile.

## Requirements
<Warning>This firmware device has specific configuration requirements for the **Main PC**. Please ensure you meet the following requirements to avoid various bugs.</Warning>
* Intel VMX/VTD must be turned `OFF` in BIOS
* AMD SVM/IOMMU must be turned `OFF` in BIOS
* DMA Kernel Protection must be turned `OFF` in BIOS and Windows
