## Common Setup Steps

<Steps>
  1. Install required software
     Ensure all necessary drivers and utilities are installed
     
  2. Configure game settings
     Adjust in-game settings for optimal performance
     
  3. Load game profile
     Import or configure game-specific profiles
     
  4. Test configuration
     Verify all features are working as expected
</Steps>

<Accordion title="License HWID Reset">

4. Reactivate on new hardware
