# Arena Breakout Configuration

## Quick Setup

<Steps>
  1. Initial Setup
     Launch S2 Loader with administrator rights
     
  2. Game Selection
     Choose Arena Breakout from the list
     
  3. Profile Setup
     Configure your preferred settings
     
  4. Game Launch
     Start Arena Breakout through game launcher
</Steps>

    - Shadows: Low
    - Textures: Medium
    - Effects: Low
    - View Distance: High
