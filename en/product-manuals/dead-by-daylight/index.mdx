# Dead by Daylight Configuration

## Quick Setup

<Steps>
  1. Initial Setup
     Start S2 Loader with administrator privileges
     
  2. Game Selection
     Choose Dead by Daylight from the list
     
  3. Unlocker Integration
     Configure unlocker settings if using
     
  4. Launch Game
     Start Dead by Daylight through Steam
</Steps>

## Performance Settings

<Tabs>
  <Tab title="Graphics">
    ### Optimal Settings
    - Resolution: 1080p recommended
    - Shadow Quality: Low
    - Anti-Aliasing: Low
    - Texture Quality: Medium
    - Post-Processing: Low
  </Tab>
  
  <Tab title="Game Settings">
    ### In-Game Options
    - VSync: Off
    - Motion Blur: Off
    - Auto-Adjust: Off
    - FPS Cap: 120
  </Tab>
</Tabs>

## License Management

<Accordion title="HWID Reset">
### Reset Process
1. Access account dashboard
2. Select DBD license
3. Request HWID reset
4. Wait for confirmation
5. Reactivate license

<Note>
Keep track of your reset cooldown period
</Note>
</Accordion>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Game Crashes">
    - Verify game files
    - Update graphics drivers
    - Check EAC status
    - Clear game cache
  </Accordion>
  
  <Accordion title="Unlocker Issues">
    - Update unlocker
    - Check compatibility
    - Verify file paths
    - Reset configuration
  </Accordion>
  
  <Accordion title="Performance Problems">
    - Lower graphics settings
    - Update system drivers
    - Check DMA connection
    - Monitor system resources
  </Accordion>
</AccordionGroup>
