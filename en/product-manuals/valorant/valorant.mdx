# VALORANT Configuration

## Important Notice

<Warning>
Always verify firmware compatibility before use. VALORANT's anti-cheat system requires specific firmware versions.
</Warning>

## Quick Setup

<Steps>
  1. Firmware Check
     Verify your firmware is compatible with current VALORANT version
     
  2. Launch Setup
     Start S2 Loader as administrator
     
  3. Select Game
     Choose VALORANT from the game list
     
  4. Configure Settings
     Apply recommended configuration
</Steps>

## Performance Optimization

<Tabs>
  <Tab title="Video Settings">
    ### Recommended Settings
    - Resolution: Native
    - Material Quality: Low
    - Texture Quality: Low
    - UI Quality: Low
    - Vignette: Off
    - VSync: Off
    - Anti-Aliasing: MSAA 2x
  </Tab>
  
  <Tab title="System Settings">
    ### Windows Optimization
    - Disable Full-screen Optimization
    - Set High Performance Power Plan
    - Disable Background Apps
    - Update Graphics Drivers
  </Tab>
</Tabs>

## License Management

<Accordion title="HWID Reset">
### Reset Instructions
1. Visit account management
2. Locate VALORANT license
3. Request HWID reset
4. Wait for processing
5. Reactivate license

<Warning>
Limited resets available per month
</Warning>
</Accordion>
