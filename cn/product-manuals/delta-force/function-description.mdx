## ① 切换 国际服 / 国服支持
<Warning>
  产品默认为国服模式。想要切换至国际服需打开国际服模式并重启菜单。
</Warning>
<Check>
  打开菜单后 前往 `杂项` -> 启用 `国际服` 选项  
    
  ⚠️  打开后请务必重开菜单才能奏效
</Check>

## ② Physx 模型
<Note>
  Physx模型是一个强大的掩体判断模型，启用后将大大优化于自瞄和绘制的掩体判断。  
  产品默认为并不启用Physx模型。需要手动开启
</Note>
<Check>
  打开菜单后 前往 `杂项` -> 启用 `Physx` 下的开关选项  

  ⚠️ 打开Physx选项开关即可，绘制模型开关为debug调试功能。不需要开启。  
  ⚠️ 打开后请务必重开菜单才能奏效。
</Check>

## ③ 产品绘制分辨率调节
<Check>
  打开菜单后 前往 `杂项` -> 根据屏幕分辨率调节 `屏幕宽度` 和 `屏幕高度`
</Check>
