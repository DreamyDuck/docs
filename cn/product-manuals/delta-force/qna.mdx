## ① 切换 国际服 / 国服支持
<Warning>
  产品默认为国服模式。想要切换至国际服需打开国际服模式并重启菜单。
</Warning>
<Check>
  打开菜单后 前往 `杂项` -> 启用 `国际服` 选项  
    
  ⚠️  打开后请务必重开菜单才能奏效
</Check>

## ② 优化自瞄和绘制掩体判断
<Warning>
  默认使用Physx为关闭。用户可自行开启以使用该绘制掩体判断模型。此绘制模型吃电脑性能，不推荐低配辅机启用该选项。
</Warning>
<Check>
  打开菜单后 前往 `杂项` -> 关闭 `Physx` 和 `绘制模型`  
    
  ⚠️  打开Physx选项开关即可，绘制模型开关为debug调试功能。不需要开启。  
  ⚠️  打开后请务必重开菜单才能奏效
</Check>

## ③ 拖框 / 闪退 / 关闭Physx
<Warning>
  拖框，卡顿，闪退等问题通常最大的问题是Physx模型吃了大量资源。
</Warning>
<Check>
  打开菜单后 前往 `杂项` -> 关闭 `Physx` 和 `绘制模型`
</Check>
<Check>
  打开菜单后 前往 `杂项` -> 点击 `重载战局`
</Check>

## ④ 产品绘制分辨率畸形 / 无法正确识别屏幕分辨率 / 错误绘制位置
<Check>
  打开菜单后 前往 `杂项` -> 根据屏幕分辨率调节 `屏幕宽度` 和 `屏幕高度`
</Check>
