# Apex Legends Instruction

<InjectSteps/>

## Menu Language

<Accordion title="Changing Interface Language">
### Language Options
1. Open the S2 Loader settings
2. Navigate to Language Settings
3. Select your preferred language
4. Restart the loader to apply changes

<Note>
Available languages include English, Chinese, Russian, and more
</Note>
</Accordion>

## License Management

<Accordion title="HWID Reset">
### Reset Steps
1. Visit the [License Portal](https://scarlet.technology/account)
2. Navigate to License Management
3. Select Apex Legends license
4. Click "Reset HWID"
5. Wait 24 hours before next reset

<Warning>
Limited to one reset per 24-hour period
</Warning>
</Accordion>

## Troubleshooting

### Common Issues

<AccordionGroup>
  <Accordion title="Connection Errors">
    - Verify S2 Loader is running as administrator
    - Check firewall settings
    - Ensure anti-virus is not blocking connection
  </Accordion>
  
  <Accordion title="Performance Issues">
    - Update graphics drivers
    - Verify DMA connection speed
    - Check system requirements
  </Accordion>
  
  <Accordion title="Game Crashes">
    - Verify game file integrity
    - Update to latest game version
    - Check for Windows updates
  </Accordion>
</AccordionGroup>
