# The Finals Configuration

## Important Notice

<Warning>
The Finals uses advanced anti-cheat systems. Always ensure you're using the latest firmware version and following all safety guidelines.
</Warning>

## Quick Setup

<Steps>
  1. Firmware Verification
     Check and update to latest firmware version
     
  2. Launch Setup
     Start S2 Loader as administrator
     
  3. Game Selection
     Choose The Finals from game list
     
  4. Configuration
     Apply recommended settings
</Steps>
