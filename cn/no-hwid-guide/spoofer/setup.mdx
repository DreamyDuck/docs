# 开始之前
1. 卸载计算机上的所有防病毒软件，包括 Avast、Malwarebytes、360、鲁大师等。  
2. 禁用计算机上的内置防病毒软件。（Windows Defender）
3. 使用 `DX 修复工具`。
4. 关闭并退出所有叠加层软件，如 `GAME++`、`MSI Afterburner`、`NVIDIA GeForce`、`OneDrive` 等。  
5. 关闭并退出计算机上的任何调试器，如 `CheatEngine`、`IDA`、`Scylla` 等。
6. 卸载计算机上的任何模拟器，如 `腾讯模拟器`、`Gameloop 模拟器`、`雷电模拟器` 等。
