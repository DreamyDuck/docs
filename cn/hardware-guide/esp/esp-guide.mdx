# ESP 远程访问解决方案

## 概述

<Note>
我们的 ESP 解决方案通过 Moonlight 和 Parsec 提供远程游戏访问，提供低延迟流媒体，无需使用融合器。
</Note>

## 解决方案选项

<CardGroup>
  <Card title="Moonlight" icon="moon">
    - 低延迟流媒体
    - NVIDIA 优化
    - 本地网络重点
    - 高质量输出
  </Card>
  
  <Card title="Parsec" icon="globe">
    - 云游戏支持

    - 协作功能
  </Card>
</CardGroup>
