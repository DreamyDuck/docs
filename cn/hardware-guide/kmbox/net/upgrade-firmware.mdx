# ➁ 升级您的固件
<Note>
    请仔细按照说明操作。确保您的 COM 端口号正确。如果全部失败，请尝试使用另一台 PC。
</Note>
1. 拔掉所有 USB 线缆以断开设备连接。
2. 按住设备内部的唯一按钮并保持。
3. 在按住上述按钮的同时，将 USB 线缆插入端口**（游戏 PC）**。您将看到指示灯快速闪烁。现在可以松开按钮。
4. 打开升级工具 `WCHMcuIAP_WinAPP.exe`。（双击）
<Frame>
      <img src="/cn/images/net8.png" />
</Frame>

<Note>
    选择您想要升级的固件，然后点击下载。
</Note>

5. 如果下载成功，设备将自动重启。此时显示正常
<Frame>
      <img src="/cn/images/net9.png" />
</Frame>

<Warning>
    如果失败，请重试上述步骤
</Warning>
