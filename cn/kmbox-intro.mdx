## 什么是 Kmbox？

**KMbox**（键盘和鼠标模拟盒）是一种在 **DMA（直接内存访问）作弊设置** 中常用的硬件设备，用于在规避反作弊检测的同时自动化或增强游戏中的键盘/鼠标输入。与基于软件的输入工具不同，KMbox 在硬件层面模拟物理外设，使 **EasyAntiCheat**、**BattlEye** 或 **Vanguard** 等反作弊系统更难检测到可疑活动。

## 在 DMA 作弊中的角色

### 硬件级输入模拟
KMbox 作为**第二台 PC（作弊 PC）**（运行作弊软件）和**游戏 PC** 之间的桥梁。它将数字命令（如后坐力控制脚本、快速射击宏）转换为硬件级键盘/鼠标信号，使输入看起来像是来自合法外设。

### 与 DMA 集成
- **DMA 卡**从游戏 PC 读取游戏内存数据（如玩家位置、武器状态）。
- 作弊 PC 处理这些数据并向 KMbox 发送指令，KMbox 在游戏 PC 上执行输入。
- 这种分离确保游戏 PC 从不直接运行作弊软件，降低检测风险。

## 主要功能

| 功能 | 描述 |
|---------|-------------|
| **反检测输入** | 模拟人性化的鼠标移动和按键，绕过软件级反作弊扫描。 |
| **宏脚本** | 支持自定义脚本用于后坐力控制、快速射击、自动拾取或建造（如 Fortnite）。 |
| **即插即用** | 无需驱动程序；通过 USB 直通或网络（Wi-Fi/ETH 型号）工作。 |
| **跨平台** | 通过适配器兼容 PC、PlayStation、Xbox 甚至移动设备。 |
| **低延迟** | 实时输入执行响应时间低于 1ms（对竞技游戏至关重要）。 |

## 常见用途

- **后坐力控制**：自动调整鼠标移动以抵消武器后坐力模式（如 *CS2*、*Apex Legends*）。
- **自瞄辅助**：平滑瞄准移动朝向目标（与基于 DMA 的 ESP 数据配合使用时）。
- **快速射击**：绕过半自动武器的射速限制。
- **自动化操作**：触发重复任务的按键（如拾取、建造）。

## 在 DMA 作弊系统中的设置

### 硬件连接
- **游戏 PC** → 通过 USB 连接到 KMbox。
- **KMbox** → 通过 USB、Wi-Fi 或以太网（取决于型号）连接到**第二台 PC（作弊 PC）**。
- **DMA 卡** → 安装在**主 PC** 中以读取游戏 PC 内存。

### 软件配置
- 将自定义脚本（如 Lua/Python）加载到 KMbox 的固件中。
- 将 DMA 数据（如敌人坐标）与输入脚本同步以实现动态调整。

### 示例工作流程
- DMA 卡通过内存读取检测敌人位置。
- 作弊 PC 计算最佳瞄准轨迹。
- KMbox 将其转换为游戏 PC 上的鼠标移动。

## 相比软件工具的优势

- **设计上不可检测**：在硬件层面运行，避免内存扫描或内核级反作弊检查。
- **多功能性**：与仅支持 DMA 的设置不同，可与游戏机和 PC 配合使用。
- **自定义**：脚本可以针对特定游戏或游戏风格进行微调。

## 限制

- **复杂性**：需要脚本知识才能最大化效果。
- **固件风险**：KMbox 固件不是开源的，因此无法获取自定义固件。（官方每月提供例行固件更新）

## 流行的 KMbox 型号

1. **KMbox A**：非常老的版本，用于基本脚本控制。
2. **KMbox B + / B Pro+**：上市超过三年，由于使用 COM 端口，该产品最近已被多个反作弊系统检测到。
3. **KMbox Network**：基于 Kmbox B+ 升级，已修改为使用网络端口控制主机，与 B 系列相比更安全稳定。
4. **KMbox NVideo**：基本上是 Kmbox Network + 内置采集卡。更常用于 AI 产品。

## 与类似工具比较

| 工具 | 检测风险 | 输入自定义 | DMA 兼容性 |
|------|----------------|---------------------|-------------------|
| **KMbox** | 极低 | 高 | 是 |
| **Arduino** | 低 | 中等 | 是 |
| **AutoHotkey** | 高 | 高 | 否 |

总结来说，**KMbox** 是高级 DMA 作弊设置中的关键组件，实现不可检测的硬件级输入自动化。当与 DMA 内存读取和作弊 PC 配合使用时，它提供了一个无需直接软件交互即可获得竞争优势的强大解决方案。
