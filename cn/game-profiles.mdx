# Game Profiles

Game profiles allow you to customize and save settings for specific games, making it easy to switch between different configurations.

## What are Game Profiles?

Game profiles are saved configurations that include:

- Memory addresses for specific game features
- Custom keybindings
- Feature toggles and settings
- Visual preferences

By creating game profiles, you can quickly switch between different configurations without having to manually adjust settings each time you play a different game.

## Creating a Game Profile

<Steps>
  <Step title="Open the Scarlet application">
    Launch the Scarlet application on your computer and ensure your device is connected.
  </Step>
  <Step title="Navigate to Profiles">
    Click on the "Profiles" tab in the main navigation.
  </Step>
  <Step title="Create a new profile">
    Click the "+" button to create a new profile.
  </Step>
  <Step title="Select a game">
    Choose a game from the dropdown menu or select "Custom" to create a profile for an unlisted game.
  </Step>
  <Step title="Configure settings">
    Adjust the settings according to your preferences:
    - ESP settings (colors, distance, etc.)
    - Aimbot settings (smoothness, FOV, etc.)
    - Keybindings
    - Visual preferences
  </Step>
  <Step title="Save the profile">
    Give your profile a name and click "Save".
  </Step>
</Steps>

## Managing Profiles

### Switching Profiles

To switch between profiles:

1. Open the Scarlet application
2. Go to the "Profiles" tab
3. Select the profile you want to use
4. Click "Activate"

### Editing Profiles

To edit an existing profile:

1. Open the "Profiles" tab
2. Select the profile you want to edit
3. Click the "Edit" button
4. Make your changes
5. Click "Save"

### Sharing Profiles

You can share your profiles with other Scarlet users:

1. Select the profile you want to share
2. Click the "Export" button
3. Choose a location to save the profile file (.scarlet)
4. Share the file with other users

To import a profile:

1. Click "Import" in the Profiles tab
2. Select the .scarlet file
3. The profile will be added to your list

## Advanced Profile Configuration

<Accordion title="Memory Address Customization">
  Advanced users can manually configure memory addresses for game features:
  
  ```json
  {
    "player_base": "0x12345678",
    "health_offset": "0x123",
    "position_offset": "0x456",
    "weapon_offset": "0x789"
  }
  ```
  
  These addresses can be entered in the "Advanced" tab when editing a profile.
</Accordion>

<Accordion title="Custom Scripts">
  Scarlet supports custom Lua scripts for advanced functionality:
  
  ```lua
  function OnPlayerDetected(player)
    if player.team ~= localPlayer.team then
      -- Custom logic for enemy players
      DrawBox(player.position, player.color)
    end
  end
  
  function OnTick()
    -- Code that runs every frame
  end
  ```
  
  Scripts can be added in the "Scripts" tab of the profile editor.
</Accordion>

## Troubleshooting

If you encounter issues with your profiles:

- Ensure you're using the latest version of the Scarlet software
- Verify that the game version matches the profile version
- Check that your Scarlet device firmware is up to date
- Try resetting the profile to default settings
- Consult our [Discord community](https://discord.gg/scarlet) for assistance
