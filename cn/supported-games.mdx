## Compatibility Notes

- Game support may vary based on game updates and anti-cheat changes
- Some features may require specific Scarlet hardware models
- Console support requires additional hardware configuration

## Request Game Support

Don't see your favorite game on the list? Request support for additional games by:

1. Joining our [Discord community](https://discord.gg/scarlet)
2. Submitting a game support request in the #game-requests channel
3. Providing details about the game and desired features

Our development team regularly reviews requests and prioritizes games based on community demand.
